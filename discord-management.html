Moderation Settings

How To Use Settings Correctly

  1. Step 1: Pick a profile in Quick Setup and click Apply Profile.
  2. Step 2: Set Log Channel ID, then enable Moderation Enabled.
  3. Step 3: Add Exempt Role IDs and Exempt Channel IDs for trusted staff/bot channels.
  4. Step 4: Save config, monitor cases for a day, then open Advanced Settings only if needed.
